# 🌍 Data-Driven Analysis & Prediction of Indian Climate Patterns (2024–2025)

## 📌 Overview
This project focuses on analyzing and predicting climate patterns across India using **Data Science and Machine Learning techniques**.

It explores important environmental factors like temperature, rainfall, and humidity to uncover trends and build predictive models for future climate conditions.

---

## 📂 Dataset
- **File:** `Indian_Climate_Dataset_2024_2025.csv`

The dataset includes:
- Temperature (°C)
- Rainfall (mm)
- Humidity (%)
- Other climate-related attributes

---

## 🛠 Technologies Used
- Python 🐍  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

## 📊 Project Workflow
1. Data Collection  
2. Data Cleaning & Preprocessing  
3. Exploratory Data Analysis (EDA)  
4. Data Visualization  
5. Model Building (Linear Regression)  
6. Model Evaluation  

---

## 📈 Key Features
- 📊 Climate data analysis  
- 📉 Trend visualization  
- 🤖 Predictive modeling  
- 📏 Model evaluation using:
  - Mean Absolute Error (MAE)
  - R² Score
